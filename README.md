# Court-Data Fetcher & Mini-Dashboard

## ✅ Target Court
Faridabad District Court (https://districts.ecourts.gov.in/faridabad)

## ⚙️ Tech Stack
- Python (Flask)
- Playwright (Headless scraping)
- SQLite (Storage)
- Bootstrap (UI)

## 🧩 CAPTCHA Handling

```bash
pip install -r requirements.txt

python run.py
